通用属性

属性编号 属性字节数 属性描述
0 W 面世日期，从1920年算起，每晚一天加一
2 B reliability decay speed
3 B 车辆寿命
4 B 多少年内游戏中可购买该车辆
6 B climate availability
7 B load amount

8 B Speed in mph*3.2，速度

0F B Capacity，容量

GRFv≥8 For GRF version 8 and above: Type A '(translated) cargo bit'
11 B Cost factor，成本
12 B Sound effect: 17/19/1A for regular, 3C/3E for toyland
13 B Power in 10 hp, see below，动力

15 B Speed in mph*0.8, see below

18 B Coefficient of tractive effort
19 B Coefficient of air drag

20 B* Sort the purchase list
21 B Visual effect
22 W Custom cargo ageing period
23 B Make vehicle shorter, see train property 21
24 B n*B List of always refittable cargo types, see train property 2C
25 B n*B List of never refittable cargo types, see train property 2D

属性编号 属性字节数 属性描述
5 B Track type (see below)

9 W Speed in mph*1.6 (see below)，速度
0B W Power (0 for wagons)，动力
0D B Running cost factor (0 for wagons)，运营成本
0E D Running cost base, see below
12 B Sprite ID (FD for new graphics)
13 B Dual-headed flag; 1 if dual-headed engine, 0 otherwise
14 B Cargo capacity，容量
15 B Cargo type, see CargoTypes
GRFv≤7 For GRF version 7 and below: Type B 'cargo slot'
GRFv≥8 For GRF version 8 and above: Type A '(translated) cargo bit'
16 B Weight in tons，重量
17 B Cost factor，成本
18 B Engine rank for the AI (AI selects the highest-rank engine of those it can buy)
19 B Engine traction type (see below)
1A B/B*[2] Not a property, but an action: sort the purchase list.
1B W Power added by each wagon connected to this engine, see below
1C B Refit cost, using 50% of the purchase price cost base
1D D Bit mask of cargo types available for refitting, see column 2 (bit value) in CargoTypes
1E B Callback flags bit mask, see below
1F B Coefficient of tractive effort
20 B Coefficient of air drag
21 B Make vehicle shorter by this amount, see below
22 B Set visual effect type (steam/smoke/sparks) as well as position, see below
23 B Set how much weight is added by making wagons powered (i.e. weight of engine), see below
24 B High byte of vehicle weight, weight will be prop.24*256+prop.16
25 B User-defined bit mask to set when checking veh. var. 42
26 B Retire vehicle early, this many years before the end of phase 2 (see Action0General)
27 B Miscellaneous flags
28 W Refittable cargo classes
29 W Non-refittable cargo classes
2A D Long format introduction date
2B W Custom cargo ageing period
2C B n*B List of always refittable cargo types
2D B n*B List of never refittable cargo types
 

飞机属性

属性编号 属性字节数 属性描述
8 B Sprite (FF for new graphics)
9 B Is helicopter? 2=no, 0=yes
0A B Is large? 0=no, 1=yes (i.e. can't safely land on small airports)
0B B Cost factor，成本
0C B Speed in units of 8 mph, that is: property = (speed in mph) / 8，速度
0D B Acceleration in units of 3/8 mph/tick, that is: property = (acceleration in mph/tick) * 8/3 [1]
0E B Running cost factor，运营成本
0F W Primary cargo capacity (passenger or refitted cargo)，乘客、邮件容量
11 B Secondary cargo capacity (mail)，货物容量
12 B Sound effect
13 D Bit mask of cargo types available for refitting, see column 2 (Bit Value) in CargoTypes
14 B Callback flags bit mask, see below
15 B Refit cost, using 1/32 of the default refit cost base
16 B Retire vehicle early, this many years before the end of phase 2 (see Action0General)
17 B Miscellaneous vehicle flags
18 W Refittable cargo classes, see train prop. 28
19 W Non-refittable cargo classes, see train prop. 29
1A D Long format introduction date
1B B* Sort the purchase list
1C W Custom cargo ageing period
1D B n*B List of always refittable cargo types, see train property 2C
1E B n*B List of never refittable cargo types, see train property 2D
1F W Aircraft range in tiles. Distance is euclidean, a value of 0 means range is unlimited
